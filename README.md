# 🌐 Sk Aman | Portfolio

🚀 A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my projects, skills, and experience.

👉 https://skaman-portfolio.vercel.app

## 📌 Features

* 📱 Responsive Design: Fully optimized for Mobile, Tablet, and Desktop views.
* 🧩 Project Showcase: Interactive gallery of my web and backend applications.
* 📄 Resume Integration: Direct link to my professional resume (PDF) for easy viewing.
* 🎨 Modern UI: Clean, minimalist interface with smooth transitions.
* ⚡ Optimized Assets: Uses SVGs for crisp icons and fast load times.
* 📞 Contact section

* Frontend: HTML5, CSS3, JavaScript (ES6+)

* Assets: Optimized SVGs & High-quality PDF Resume

* Hosting: Vercel

* Version Control: Git & GitHub

## ⚙️ Local Development

1. Clone the repository:

```
git clone https://github.com/aman4github/aman-portfolio.git
```

2. Navigate to the folder:

```
cd aman-portfolio
```

3. Open with VS Code:

```
code .
```

4. Launch Live Server:

* Right-click index.html and select "Open with Live Server".
